This project introduces basic decision-making in Python using `if`, `elif`, and `else` statements. It is designed for 
beginners learning how robots (or programs) make choices based on conditions.

## What You’ll Learn
- How `if` statements work with True/False conditions  
- Proper indentation and why it matters  
- Comparison operators (`==`, `!=`, `<`, `>`, `<=`, `>=`)  
- Using `if / elif / else` for multiple choices  
- Combining conditions with `and` / `or`  
- Converting between strings and numbers  

## Example
```python
distance = 15

if distance < 10:
    print("Stop")
elif distance < 20:
    print("Slow down")
else:
    print("Keep moving")
